Comandos

Mi primer repositorio

mkdir myrepo # md for win
cd myrepo
git init

Mi primer commit

vim Readme # (mate, nano, edit...)
git status
git add
git commit -m 'Add readme file'

Modifiquemos Readme

vim Readme # (mate, nano, edit...)
git status

Añadamos el cambio a stage

git add
git status

Confirmemos el cambio

git commit
git status

Branches / ramas

git checkout -b test
git branch -a
git status

Haz al menos dos commits en la branch test

¿Qué pasa con master?

git checkout master
git log
git checkout test
git log

Merge / Unir

git checkout master
git merge test
git log

Borrar una branch

git branch -d test

Workflow

  • Crear una topic-branch

  • Merge master en la topic branch
  • Merge topic en master
  • Borrar la topic branch
